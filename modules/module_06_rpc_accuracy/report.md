# 实验报告

## 一、实验目的

本模块是《卫星摄影测量》综合编程作业的最终环节，负责 RPC（有理多项式系数）模型的参数解算与精度评定。

RPC 模型是一种通用的卫星影像几何模型，通过有理多项式将地面大地坐标 (lon, lat, height) 映射到影像像方坐标 (row, col)，无需直接使用卫星轨道和姿态参数。本模块在模块 05 生成的样本点数据基础上，完成以下目标：

1. 建立 RPC 有理多项式的线性化最小二乘方程。
2. 求解归一化参数（OFF / SCALE）和 78 个多项式系数（行/列方向各 39 个）。
3. 内符合精度检查：训练样本回代检验 RPC 拟合精度。
4. 外符合精度检查：独立检查点或 DEM 空间采样检验 RPC 泛化精度。
5. 综合精度报告：汇总内/外符合精度指标。

## 二、实验原理

### 2.1 RPC 模型定义

RPC 模型使用三次齐次多项式的比值来建立地面坐标与像方坐标的映射关系：

```
row  = NUM_row(P, L, H) / DEN_row(P, L, H) * LINE_SCALE + LINE_OFF
col  = NUM_col(P, L, H) / DEN_col(P, L, H) * SAMP_SCALE + SAMP_OFF
```

其中 (P, L, H) 为归一化后的大地坐标：

```
P = (lat - LAT_OFF) / LAT_SCALE
L = (lon - LONG_OFF) / LONG_SCALE
H = (height - HEIGHT_OFF) / HEIGHT_SCALE
```

### 2.2 多项式基函数

RPC 使用 20 个三次齐次多项式基函数：

- 1 个常数项：1
- 3 个一次项：L, P, H
- 6 个二次项：LP, LH, PH, L^2, P^2, H^2
- 10 个三次项：LPH, L^3, LP^2, LH^2, L^2P, P^3, PH^2, L^2H, P^2H, H^3

### 2.3 线性化最小二乘求解

RPC 正变换的有理形式 `r = NUM / DEN` 可线性化为：

```
NUM - r * DEN = 0
```

展开后得到：

```
[B | -r * B_{1:19}] * [num_0..19; den_1..19] = r * B_0
```

其中 B_0 = 1 固定为分母首项（避免平凡解），每方向求解 39 个未知系数（20 个分子 + 19 个分母）。行、列方向独立求解。

### 2.4 归一化处理

为改善最小二乘方程的数值条件，所有坐标在求解前需归一化到 [-1, 1] 范围：

- OFF = 均值（使归一化后以零为中心）
- SCALE = 最大偏差（使归一化后不超过 [-1, 1]）

### 2.5 精度评定方法

**内符合精度**：将训练样本点代入 RPC 正变换，与参考像方坐标比较残差。反映 RPC 模型对训练数据的拟合能力。

**外符合精度**：
- **独立检查点法**：用未参与训练的独立检查点，比较 RPC 正变换结果与参考坐标。反映模型泛化能力。
- **DEM 空间采样法**：从 DEM 在 RPC 覆盖范围内均匀采样地面点，用 RPC 正变换分析像方坐标分布。配合严格成像模型（模块 04）的反投影结果，可完成完整的外符合精度评定。
- **严格模型对比法**：用 DEM 采样点的严格模型反投影坐标作为参考，与 RPC 正变换结果比较残差。

**统计指标**：
- `row_error` = row_rpc - row_ref（行方向误差）
- `col_error` = col_rpc - col_ref（列方向误差）
- `plane_error` = sqrt(row_error^2 + col_error^2)（平面误差）
- RMSE（均方根误差）、MAE（平均绝对误差）、MAX（最大绝对误差）

## 四、实验步骤

### 4.1 RPC 解算

1. 读取模块 05 输出的 RPC 样本点数据。
2. 调用 `solve_rpc()` 计算归一化参数并求解多项式系数。
3. 调用 `save_rpc_result()` 将 RPC 参数保存为文本文件。

### 4.2 内符合精度检查

4. 将训练样本点代入 RPC 正变换。
5. 与参考像方坐标比较，计算行/列/平面误差。
6. 统计 RMSE、MAE、MAX 指标。

### 4.3 外符合精度检查

7. 读取独立检查点数据。
8. 调用 `evaluate_accuracy()` 计算外符合精度。
9. （可选）从 DEM 采样覆盖范围内地面点，分析 RPC 正变换像方分布。
10. （可选）配合严格模型反投影结果，完成完整外符合检验。

### 4.4 综合报告

11. 调用 `comprehensive_accuracy_report()` 汇总内/外符合精度。
12. 保存精度评定结果到 CSV 文件。

## 五、核心代码说明

### 5.1 rpc_solver.py

- `_compute_basis_matrix()`: 构建 (N, 20) 多项式基函数矩阵。
- `_compute_normalization_params()`: 从样本点分布计算 10 个归一化参数。
- `_solve_coefficients()`: 构建线性化方程并调用 `numpy.linalg.lstsq` 求解。
- `solve_rpc()`: 主函数，串联归一化、求解、内符合检验流程。
- `rpc_forward()`: RPC 正变换，大地坐标 -> 像方坐标。
- `save_rpc_result()`: 将参数写入标准格式文本文件。

### 5.2 accuracy_eval.py

- `_compute_statistics()`: 计算误差统计指标（RMSE, MAE, MAX, mean, std）。
- `parse_rpc_file()`: 解析 RPC 参数文件（来自 rpc_solver）。
- `rpc_forward()`: RPC 正变换（来自 rpc_solver）。
- `evaluate_accuracy()`: 检查点精度评定（内/外符合通用）。
- `sample_dem_points()`: 从 DEM 在 RPC 覆盖范围内均匀采样地面点。
- `external_accuracy_check()`: 基于 DEM 的外符合精度检查（独立实现）。
- `external_accuracy_with_strict_model()`: 配合严格模型的外符合精度检查。
- `comprehensive_accuracy_report()`: 综合精度报告，汇总内/外符合精度。

## 七、问题与改进

### 当前问题

1. **测试数据过于简单**：模块 05 的样本数据使用了简化的线性正投影模型，无法充分检验 RPC 模型对非线性几何变形的拟合能力。在实际卫星影像上，RPC 拟合残差通常在亚像素量级。

2. **外符合精度需要严格模型配合**：DEM 空间采样提供了覆盖范围内的独立地面点，但完整的外符合精度评定需要严格成像模型（模块 04）提供参考像方坐标。单独运行时仅能输出 DEM 采样点的 RPC 正变换分布。

### 改进方向

1. 在最终联调时，使用严格成像模型生成的虚拟 GCP 进行解算，并用 DEM 采样的独立检查点进行外符合精度评定。
2. 可增加 Tikhonov 正则化（岭回归）改善法方程条件数，尤其在样本点分布不均匀时。
3. 可增加逐点残差输出和可视化功能，便于分析误差空间分布规律。
